简介
 

Home Assistant的组件


此Home Assistant的组件可以让小度对接HomeAssistant平台上的设备,控制设备、查询上报设备状态。

注意：本插件只负责上报HomeAssistant的设备状态信息到服务器，控制功能需要在平台上配置。
